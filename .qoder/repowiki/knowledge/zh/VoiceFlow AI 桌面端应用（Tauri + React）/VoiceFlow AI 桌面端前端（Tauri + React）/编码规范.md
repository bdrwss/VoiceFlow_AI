- 所有与 Rust 后端的交互统一通过 `@tauri-apps/api/core` 的 `invoke` 函数发起，并以字符串命令名（如 `download_sensevoice`、`simulate_typing`、`replace_with_ai_text`）作为通道。
- 跨窗口通信采用 Tauri 事件模型：主窗口通过 `WebviewWindow.getByLabel('indicator').emit(...)` 向前台小药丸广播状态，小药丸则通过 `listen('pill-action', ...)` 回发取消/提交动作。
- 异步副作用一律在 `useEffect` 中注册，并在返回的清理函数中调用 `unlisten()` 或 `clearInterval`，防止组件卸载后残留监听器。
- 长耗时操作（模型初始化、下载进度）通过 Tauri 事件（`download-progress`、`shortcut-state`）推送，前端用 `useState` 映射到 UI 进度条而非轮询。
- 设置项与历史记录均先落盘 `localStorage`（key 前缀 `vf_`），再按需 `invoke` 同步至 Rust 后端，保证离线可用与重启一致。
