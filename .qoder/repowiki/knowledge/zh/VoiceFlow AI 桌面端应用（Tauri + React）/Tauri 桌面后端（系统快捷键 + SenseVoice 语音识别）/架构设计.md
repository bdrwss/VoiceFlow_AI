模块采用标准 Tauri v2 双 crate 布局：`src/main.rs` 仅作为入口调用 `tauri_app_lib::run()`，业务逻辑集中在 `src/lib.rs` 中通过 `tauri::Builder` 组装；`build.rs` 使用 `tauri_build::build()` 生成上下文。

- 进程内状态：`AppState` 以 `RwLock<String>` / `RwLock<Vec<String>>` 持有当前监听键与黑名单，通过 `State<'_, AppState>` 注入到所有 `#[tauri::command]`。
- 系统交互层：`rdev::listen` 在独立线程阻塞监听全局按键，匹配后通过 `AppHandle::emit("shortcut-state", ...)` 推送事件；`enigo` + `arboard` 实现跨平台 Ctrl/Cmd+V 粘贴与逐字 Backspace 替换。
- 能力边界：`mod sensevoice` 单独封装 ASR 子功能，暴露三个异步命令 `check_sensevoice_ready` / `download_sensevoice` / `transcribe_sensevoice`，由 `lib.rs` 通过 `generate_handler!` 注册。
- 依赖方向：`main.rs → lib.rs → (sensevoice.rs, rdev/enigo/arboard, reqwest/tar/bzip2)`，前端通过 `tauri.conf.json` 中的两个窗口（`main` 控制面板 + `indicator` 悬浮岛）消费 invoke 与事件。
- 构建产物：`Cargo.toml` 将 lib 命名为 `tauri_app_lib` 并输出 `staticlib/cdylib/rlib`，release profile 开启 strip/lto/opt-level=z/panic=abort 以最小化体积。
